Volume Rendering

In this project, I have implemented a simple direct volume rendering (Figure 1).


Figure 1. Bonsai

The pipeline is as follows:

  • send the 3d data to 3d texture memory on GPU by glTexImage3D(),
  • create a bounding box,
  • bind the 3d texture by glBindTexture so that the gpu shaders can access to it,
  • and, render the bounding box.

    • Also, in the fragment shader, perform a ray casting from the eye position to the end of the object. Since a fragment shader can be called at most twice for each pixel, I use gl_FrontFacing to check whether the point is facing the camera or not. If it is facing the camera, the point is discarded. The inside/outside check can be efficiently done by checking the texture coordinates are within [0, 1] in all the axis. This approach works even if the eye position is within the object (Figure 2).
